Sara Codes!

Project Samples

These are the projects I developed using p5.js in my Spring 2024 graduate Creative Coding class.

Project 1: Digital Ghost

My first foray into creative coding! I enjoyed the experience of "blindly" creating – typing out my code and only viewing the results after inputting all the details.

Project 2: Lost Object

I enjoyed manipulating circles for this project, as well as adding details to my second p5.js sketch, particularly elements of color. Correctly identifying coordinates of each detail was something that challenged me during the creation of this object.

Project 3: Love Generator

Perfectly timed with Valentine's Day, I created a random generator that changes the color of the hearts and text with each click of the mouse. I used bezier curves for the hearts which was hard but fun!

Project 4: Memory Clock

This "clock" works by showing a different memory from a very specific time on each click of the mouse. This is a way of tracking moments in time that have since slipped away. I used if / else statements in this project to determine when each image and corresponding time would be shown.

Project 5: Hi Miss Monster!

I had so much fun with this project – it was my first time working with SVG files and it was really cool to combine those with code! My favorite part of this sketch is the eyeball – it took some time to get the boinging movement just right, but turned out really successful.

Project 6: Moiré Weaving

For this project, I played with the moiré optical illusion to create a colorful loom. Loops were implemented to create the strands of "fabric".

Project 7: A Week of Dancing to Songs

This project utilized arrays to track shapes, movements, and colors, to music I listened to throughout a week. I'm quite pleased with how the colors and overlapping transparency worked out.

Project 8: Sara Lost Her Glasses!

My semester of creative coding culminated in a personal game that documents my experience with having poor eyesight. In the game, the player "finds" objects after breaking Sara's glasses. I used a combination of photo collage and illustration to replicate a unique experience.
